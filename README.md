# 📖 Sobre o Projeto

A **Plataforma Web para Divulgação de Agenda de Bandas** foi criada com o objetivo de substituir a divulgação semanal realizada apenas por imagens e stories nas redes sociais.

Com a plataforma, cada banda possui seu próprio site, onde fãs podem consultar rapidamente:

- 📅 Agenda da semana
- 📍 Local do show
- 🕒 Horário
- 📌 Localização integrada ao Google Maps
- 📷 Fotos e identidade visual da banda

Tudo isso em uma interface moderna e responsiva.
