# 📝 Observações

Objetivo do projeto:

Criar uma plataforma profissional para divulgação da agenda semanal de bandas, oferecendo uma experiência moderna, elegante e focada em dispositivos móveis.

O sistema será vendido individualmente para bandas, permitindo que cada cliente tenha seu próprio site com identidade visual personalizada e painel administrativo para atualização da agenda.
